# MarketMind AI

MarketMind AI is a financial intelligence dashboard built with Python, Streamlit, and Yahoo Finance data.

## Features

- Real-time stock information
- Interactive stock price charts
- Financial news aggregation
- News sentiment analysis
- AI-generated market explanations
- Competitor benchmarking and comparison charts
- Market capitalization and sector analysis

## Technologies Used

- Python
- Streamlit
- yfinance
- Plotly
- Feedparser
- VADER Sentiment Analysis

## Future Development

- Portfolio tracker
- Watchlist functionality
- Technical indicators (RSI, Moving Averages)
- Earnings calendar
- LLM-powered market analysis

## Run Locally

```bash
python3 -m streamlit run app.py
```
